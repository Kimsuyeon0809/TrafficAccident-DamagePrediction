# TrafficAccident data analysis
R 언어를 활용한 대구 교통사고 데이터 분석

## 주제 선정 배경 / 데이터 소개
- data 출처 : dacon(https://dacon.io/competitions/official/236193/overview/description)
- 요일, 기상상태, 시군구, 도로형태, 노면상태 등 총 23개의 독립변수, 39,610개의 행으로구성
- 개발 환경 : google colab

## 사용 모델
- Rogistic Regression
- Decision Tree
- Random Forest
- SVM
